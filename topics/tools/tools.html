Android Tools.

The recommended tool to use for Android Development is Android Studio.
Here is what you need to know when using AndroidStudio(AS):

1. Installing you can have the stable and canary version running at the same time. You just do not need to SDKs

2. Android SDK. This has SDK platforms and SDK tools that are used by AS. Each Android SDK platform package includes both the Androdi Platform and Sources pertaing to an API level by default. Each SDK has package details like:
- x86 and _64 Intel and Google APIs.
- System images
